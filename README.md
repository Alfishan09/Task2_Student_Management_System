# 📚 Task 2: Console-Based Mini Project
## Student Management System using Java

### 📌 Description
This project was developed as part of the *Alfido Tech Java Developer Internship. It is a console-based **Student Management System* that demonstrates the application of Java fundamentals, including user input handling, conditional statements, loops, arrays, and menu-driven programming.

---

## 🎯 Objective
To build a simple console application that manages student records and provides hands-on experience with core Java programming concepts.

---

## 🛠️ Features
- ➕ Add Student Records
- 👀 View All Students
- 🔍 Search Student by ID
- ❌ Exit the Application
- 📋 Menu-Driven Interface

---

## 💻 Technologies Used
- Java
- VS Code
- JDK (Java Development Kit)

## ▶️ How to Run
1. Open the project in VS Code.
2. Open StudentManagementSystem.java.
3. Compile the program:
   bash
   javac StudentManagementSystem.java
   
4. Run the program:
   bash
   java StudentManagementSystem
   
5. Select menu options and manage student records.
